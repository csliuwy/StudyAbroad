# 2026-05-05 — 主页（Ops Console）结论与品牌 / 备案修正

## 主页（`/`，`OpsHomePage`）在做什么

StudyTour Ops 主页是**游学项目运营控制台**：先**选择或创建项目**，再在同一项目上下文中使用各模块与 API。

- **项目**：列表单选、创建（名称 + 可选目的地）、重命名、删除；依赖后端 **`GET/POST /projects`** 等；失败时提示检查本机 **8080** 或 **`VITE_API_BASE_URL`**。  
- **Modules**：例如 **Internal Information Collection**，链接为 **`/projects/:projectId/modules/internal-information-collection`**，保证带当前项目 ID。  
- **协作与内容**：**Collaboration Chat**（意图 + 触发研究）、**Proposal Upload**（上传后生成研究任务）、**Research Jobs** 列表。  
- **执行面**：**Budget / Logistics / Runbook** 快捷新增；**Publish / Sync**（Travefy、Wetu、文件导出）及发布状态列表。  
- **只读视图**：**Partner View**、**Traveler View**（按项目拉取聚合数据）。

生产环境需：**Nginx 静态站点 + `/api` 反代到后端 8080**；前端构建默认走**同源 `/api`**（见 `frontend/src/apiBase.ts`）。

## 当日修正（品牌与合规）

1. **顶栏 Logo 显示不全**  
   - 原因包含：SVG 注释中误入**不可见控制字符**（部分环境解析失败或渲染异常）、主标 **`viewBox` 未给顶部留白**（粗描边弧线/图形在圆角或缩放下易被裁切）、行内 **Logo + 文字** 受 **`line-height`** 影响导致替换元素上下被「吃掉」。  
   - 处理：重写注释与 **`viewBox`**（主标增加顶部留白）、**`preserveAspectRatio="xMidYMid meet"`**；CSS 上 **`line-height: 0`** 于品牌容器、**`object-fit: contain`** 于 **`.site-logo`**。

2. **地址栏无图标**  
   - 仅 **`favicon.svg`** 时，部分浏览器标签页**不展示 SVG**。  
   - 处理：用矢量源生成 **`favicon.ico`**（16/32/48）、**`favicon-32x32.png`**，在 **`index.html`** 中 **ICO/PNG 优先于 SVG**；补 **`apple-touch-icon.png`（192×192）** 供书签与 iOS 主屏。

3. **页脚备案**  
   - 已删除 **「京ICP备15013491号-13」** 字符串及对应链接；保留 **「京ICP备15013491号-1」**、版权与 **support@denglu.net.cn** 等其余文案。

## 部署与重启约定

- **前端静态**：执行仓库根或脚本所示流程：**`scripts/deploy-static-to-www.sh`**（构建 → 同步 **`/var/www/studyabroad`** → **`nginx -t` + reload**）。  
- **后端**：若已安装 **`studytour-backend`** systemd 单元，脚本会尝试 **`systemctl restart`**；否则需在本机按既有方式重启（例如 **`cd backend && npm run start`**）。  
- **图标二进制**：**`favicon.ico`**、**`favicon-32x32.png`**、**`apple-touch-icon.png`** 可由 **`frontend/public`** 下 SVG 经 ImageMagick **`convert`** 再生成；改标后应重新生成并再次部署。
